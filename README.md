# <a id='en_us'>BalanceSheet Repository</a>
[正體中文](#zh_tw)

- `api`: The directory where API placed in.
- `frontend`: The directory where frontend page (eg. index.html) placed in.

## Build
```
$ cd frontend
$ yarn build
$ yarn package
```

Or, just use pre-compiled packages :)
